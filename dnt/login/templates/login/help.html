FAQs & Answers

What database are you using to get the drug information?

PBS (Pharmaceutical Benefits Scheme) is the database we are currently using to retrieve the information about each drug, however we have the ability to modify the app so that it uses a different drug name database if needed.

What OCR engine are you using to extract the drug names from images?

We are currently using Tesseract but have the ability to change to other OCR engines if needed.

Where can I contact the developers to provide feedback or ask questions?

The developers can be contacted via curtincapstone@gmail.com. This email address can also be accessed on the Contact Us page.

I am uploading an image file to search with but nothing appears when I click the image open button in the file chooser, how do I fix this?

Try refreshing the browser by clicking the refresh button in top left corner of the browser or pressing f5.

When I take a picture with my mobile camera, it does not display in the correct orientation. How do I fix this?

Try taking the picture again, this time change the orientation of the camera from portrait to landscape.
